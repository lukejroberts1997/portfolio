Rotate-a-Date

(2019) My final academic project that was a dating application for android. It used Web RTC technologies to provide users with face to face interactions based on thier inputted preferences.

The application was created using Android Studio with Java. I implemented the Firebase Authentication API for creating accounts for the service and the OpenTok API to handle the microphone and camera comunication. The user creates an account and then connects to potentially match instantly.
